COMMENCEMENT OF THIS CODE

1. This Code comes into operation on the date
on which it is approved by the Council.

2. This Code is the relevant Code for all students
registered at UNISA from the commencement
date of this Code and repeals all previous
Student Disciplinary Codes,
policies and procedures
that may have previously
pertained.
